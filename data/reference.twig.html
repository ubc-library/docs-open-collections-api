Output Formats

Responses from the API are returned in JSON by default however there are a number of different formats available via both GET and POST requests.

GET Requests
For GET requests you should replace :outputFormat in the URL below with the format you want returned:
{{ data.api_url }}/collections/ [:name] /items/ [:itemId] /output-format/ [:outputFormat]

POST Requests
For POST requests you should set the Accept header to the format you want returned.
